# Background-Remove-On-Edge
Remove background for images by edge computing.

This project has the goal to remove the background of any images by edge computing on the user's device.
The user can upload the images on the page, the program then computes the background of the image and removes it, by using to user's device computing power. The user can then download the image without background.

Examples: https://huggingface.co/spaces/Xenova/remove-background-web
Models: 
- https://huggingface.co/briaai/RMBG-1.4
